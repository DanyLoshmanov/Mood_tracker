
# Mood Tracker — простой трекер настроения

Консольное приложение на Python для ведения дневника настроения.  
Помогает каждый день фиксировать, как ты себя чувствуешь, добавлять короткие заметки и просматривать историю.

Отлично подходит, если хочешь:
- следить за эмоциональным состоянием
- замечать закономерности (сезонность, влияние событий и т.д.)
- просто вести лёгкий личный дневник настроения

## Возможности

- Добавление настроения на текущий день (1–5 баллов)
- Просмотр записи за любой день
- Вывод всей истории
- Удаление записи
- Все данные сохраняются в локальный файл `mood.json`
- Полностью оффлайн, без интернета и серверов

## Шкала настроения

```
1. Очень плохо
2. Плохо
3. Нормально
4. Хорошо
5. Отлично
```

## Установка и запуск

Требуется **Python 3.6+** (ни каких дополнительных библиотек не нужно!)

```bash
# 1. Клонируй репозиторий
git clone https://github.com/tvoy-nik/mood-tracker.git
cd mood-tracker

# 2. Запусти приложение
python mood_tracker.py
```

Или просто скачай файл `mood_tracker.py` и запусти:

```bash
python mood_tracker.py
```

## Как пользоваться

После запуска появится меню:

```
1. Добавить настроение
2. Посмотреть за день
3. Показать все записи
4. Удалить запись
5. Выход
```

Всё интуитивно понятно — выбирай цифру и следуй подсказкам.

### Пример добавления записи

```
Выбери настроение:
1. Очень плохо
2. Плохо
3. Нормально
4. Хорошо
5. Отлично
Введите номер: 4
Заметка (можно пусто): Отличная погода и закончил важный проект
Настроение сохранено.
```

## Где хранятся данные?

В файле `mood.json` в той же папке, где лежит скрипт.  
Формат — обычный JSON, легко открыть и посмотреть вручную.

Пример содержимого:
```json
{
    "2025-11-24": {
        "mood": "Хорошо",
        "note": "Солнечный день и вкусный кофе"
    },
    "2025-11-23": {
        "mood": "Отлично",
        "note": "Встретился с друзьями"
    }
}
```

## Безопасность и приватность

- Никаких облачных сервисов
- Все данные только у тебя на компьютере
- Никто не увидит твои записи, если ты сам не покажешь
